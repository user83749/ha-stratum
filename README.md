# Stratum Home Assistant Add-ons

Welcome to the Stratum Home Assistant Add-on repository. This repository provides a collection of high-end dashboards for Home Assistant.

## Add-ons Included:

- **[Stratum](./ha-stratum)**: A fully customizable, premium dashboard builder for Home Assistant.

## How to use:

1. Copy this repository URL: `https://github.com/user83749/ha-stratum`
2. In Home Assistant, go to **Settings > Add-ons > Add-on Store**.
3. Click the **three dots** in the top-right and select **Repositories**.
4. Paste the URL and click **Add**.
5. You can now install **Stratum** from the list of add-ons.
